# Mercy Skins Showcase

A web showcase for Overwatch Mercy character skins. Multiple pages by category (common, legendary, OW League, Battle Pass, etc.) with a design focused on layout and visuals.

## Contents

- **index.html** — Main entry
- **comunes.html** — Common skins
- **legendarias.html** — Legendary skins
- **beneficas.html** — Charity skins
- **ow_league.html** — Overwatch League skins
- **royalknight.html** — Royal Knight
- **ba_pass.html** — Battle Pass

## Tech

HTML, CSS, Google Fonts (Montserrat).
